# 📸 Instagram Login Page (Clone)

Projeto front-end que recria a **interface de login do Instagram, desenvolvido com HTML e CSS, com foco em prática de layout, responsividade e organização de arquivos.

A aplicação simula visualmente a página inicial de login do Instagram, incluindo área de login, banners e botões para download do aplicativo.

---

## Tecnologias Utilizadas

- HTML5  
- CSS3

## Funcionalidades

- Interface inspirada na tela de login do Instagram  
- Layout organizado utilizando HTML e CSS  
- Estrutura de pastas para separar **estilos e imagens**  
- Inclusão de **favicon**  
- Botões para download do aplicativo (App Store e Google Play)

---

## Como Executar o Projeto

Clone o repositório:

git clone https://github.com/seu-usuario/LoginInstagram.git

Entre na pasta do projeto:

cd LoginInstagram

Abra o arquivo **index.html** no navegador.
